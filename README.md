Vouch
=====

About
-----

Rails 3 engines to use OAuth2 for authentication

Client
------

To use, include in your `Gemfile`

    gem "vouch", :git => "git://github.com/dpmcnevin/vouch.git"
    
Run the generator

    rails generate vouch:client
